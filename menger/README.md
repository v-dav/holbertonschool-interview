## ğŸ§ Project Overview
The Menger Sponge 2D project involves writing a function in C that draws a 2D version of the Menger sponge. The function takes the level of the Menger Sponge as input and draws it accordingly.

## ğŸ“ Problem Description
Given the level of the Menger Sponge to draw, the function should draw a 2D representation of the Menger sponge. Each level N sponge is a 3x3 square of level N-1 sponges, except for the center one, which is left empty. If the level is less than 0, the function must do nothing.

The Menger Sponge is a fractal structure that exhibits self-similarity at different levels of magnification. In this project, we aim to draw a 2D version of the Menger sponge using a function in C. The challenge involves recursively dividing a square into smaller squares and leaving the center square empty.

## ğŸ“ Prototype

```c
void menger(int level);

```

The Menger sponge is represented using the '#' character. A level 0 sponge is a simple 1x1 square.

#include <stdio.h>
#include <stdlib.h>

#include "menger.h"

/**
 * main - Entry point
 *
 * @ac: Arguments counter
 * @av: Arguments vector
 *
 * Return: EXIT_SUCCESS or EXIT_FAILURE
 */
int main(int ac, char **av)
{
    int level;

    if (ac < 2)
    {
        fprintf(stderr, "Usage: %s level\n", av[0]);
        return (EXIT_FAILURE);
    }

    level = atoi(av[1]);
    menger(level);

    return (EXIT_SUCCESS);
}
